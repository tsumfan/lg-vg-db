Ape Escape 3 is a platform video game published and developed by Sony Computer Entertainment for the PlayStation 2 video game console.

Playing as either Kei (Satoru) or Yumi (Sayaka), gameplay follows that of the previous games in which the player must catch several monkeys by using various gadgets controlled with right analogue stick. While there are fewer gadgets in this game than previous entries, a new feature in this game is the ability to morph into different forms through a device built by Aki.
