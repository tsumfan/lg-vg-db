Rupupu Cube: Lup Salad

Gameplay
PS1 Nintendo DS/DSi
Puzzle

Rupupu Cube: Lup Salad (often erroneously referred to as “Loopop Cube: Lup Salad” or “Loopop Cube: Loop Salad”) is a puzzle-game where the goal is to, as the main character Salad, clear each level of all its coloured blocks. A block disappears when matched with two or more of the same colour. Giantbomb Description
